## 코드 체계

```
[{학년접두사}{교과코드}{영역번호}-{순번}]
```

| 접두사 | 학교급/학년군 |
|--------|-------------|
| `2` | 초등 1~2학년 |
| `4` | 초등 3~4학년 |
| `6` | 초등 5~6학년 |
| `9` | 중학교 |

| 교과코드 | 교과 | 교과코드 | 교과 |
|----------|------|----------|------|
| `수` | 수학 | `사(지리)` | 사회(지리) |
| `국` | 국어 | `사(일사)` | 사회(일반사회) |
| `영` | 영어 | `역` | 역사 |
| `과` | 과학 | `기가` | 기술가정 |
| `도` | 도덕 | `생독/생프/...` | 생활외국어(언어별) |
| `체` | 체육 | `바` | 바른생활 |
| `음` | 음악 | `슬` | 슬기로운생활 |
| `미` | 미술 | `즐` | 즐거운생활 |
| `실` | 실과 | | |

### 특수 코드 형식

- **사회 괄호코드**: `[9사(지리)01-01]`, `[9사(일사)01-01]` — 세부 분야 구분
- **영어 결합레벨**: `"A B"`, `"C D"` — 두 레벨에 동일 서술 적용
- **생활외국어**: `[9생독01-01]` — 언어 약어 2자 (독/프/스/일/중/러/아/베)
