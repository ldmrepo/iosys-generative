## 1. 개요

### 1.1 프로젝트 목적

본 프로젝트는 AI 기반 차세대 문항은행 시스템을 구축하여, 문항 검색·분류·생성의 자동화 및 고도화를 달성한다.

### 1.2 최종 목표 기능

| 기능 | 설명 |
|------|------|
| 자연어 문항 검색 | 자연어 질의로 관련 문항 검색 |
| 유사 문항 검색 | 특정 문항과 유사한 문항 추천 |
| 자동 분류 | 교육과정, 성취기준, 난이도 자동 태깅 |
| 유사 문항 생성 | 기존 문항 기반 변형 문항 자동 생성 |

### 1.3 보유 자산

- 문항+이미지 데이터: 1,000건 이상
- 2022 개정 교육과정 수학 지식그래프 (기 구축)

## 2. 프로젝트 구조

### 2.1 Phase 구성

```
Phase 1 ──▶ Phase 2 ──▶ Phase 3 ──▶ Phase 4
벡터화/검색    자동분류     문항생성      통합
 (6-8주)      (4-6주)    (10-14주)    (4-6주)
```

### 2.2 Phase 간 의존성

| 선행 Phase | 후행 Phase | 의존 항목 |
|------------|------------|-----------|
| Phase 1 | Phase 2 | 임베딩 기반 클러스터 분석 결과 |
| Phase 1 | Phase 3 | 유사도 검사 인프라, Blueprint 도출 근거 |
| Phase 2 | Phase 3 | 분류 체계, 성취기준별 문항 분포 |
| Phase 1, 2, 3 | Phase 4 | 전체 기능 모듈 |

## 3. Phase별 명세

### 3.1 Phase 1: 문항 벡터화 및 검색 인프라

**목적**: 기존 문항 자산을 검색 가능한 형태로 구조화

**산출물**:

| 산출물 | 용도 |
|--------|------|
| 텍스트 임베딩 DB | 자연어 검색 |
| 이미지 임베딩 DB | 유사 이미지 검색 |
| 하이브리드 검색 API | 키워드 + 의미 검색 |
| 문항 클러스터 분석 결과 | Phase 3 Blueprint 도출 근거 |

**기술 스택**:

| 구분 | 기술 |
|------|------|
| 텍스트 임베딩 | KoSBERT / BGE-M3 |
| 이미지 임베딩 | CLIP / SigLIP |
| 벡터 DB | Milvus / Qdrant / pgvector |

**기간**: 6-8주

---

### 3.2 Phase 2: 자동 분류 시스템

**목적**: 문항 메타데이터 자동 태깅

**분류 대상**:

| 구분 | 항목 |
|------|------|
| 교육과정 | 학년, 영역, 성취기준, 핵심개념 |
| 평가 속성 | 난이도, 인지영역 |
| 문항 특성 | 문항 유형, 이미지 유형 |

**연동**: 2022 개정 교육과정 수학 지식그래프

**기간**: 4-6주

---

### 3.3 Phase 3: 이미지 참조형 문항 생성

**목적**: 이미지-텍스트 논리적 무결성이 보장된 문항 생성

**처리 흐름**:

```
이미지 입력
    │
    ▼
이미지 유형 분류
    │
    ▼
Fact 추출
    │
    ▼
Fact Graph 구축
    │
    ▼
문항 생성
    │
    ▼
Integrity Gate 검증
    │
    ├──▶ 유사도 검사 (Phase 1 인프라)
    └──▶ 자동 분류 (Phase 2 시스템)
```

**산출물**:

| 산출물 | 용도 |
|--------|------|
| Fact Graph Schema | 문항 의미 구조화 |
| Blueprint 카탈로그 | 허용 질문 유형 정의 |
| Integrity Gate Rule Set | 품질 자동 검증 |
| 유사 문항 생성 엔진 | 기존 문항 변형 |

**기간**: 10-14주

---

### 3.4 Phase 4: 통합 플랫폼

**목적**: 전 기능 통합 API 및 TEAMS 플랫폼 연동

**API 명세**:

| 엔드포인트 | 기능 |
|------------|------|
| `/search` | 자연어 / 유사 문항 검색 |
| `/classify` | 자동 분류 |
| `/generate` | 문항 생성 |

**기간**: 4-6주
