## 4. 핵심 인사이트

### 4.1 이미지 의존도 분석

```
높은 이미지 의존도 (>30%)
├── 수학 (36.1%) - 도형, 그래프, 수식 시각화
└── 과학 (33.0%) - 실험 도식, 자연현상 이미지

중간 이미지 의존도 (10-30%)
└── 사회 (13.5%) - 지도, 도표, 역사 자료

낮은 이미지 의존도 (<10%)
├── 역사 (7.9%) - 주로 텍스트 기반
├── 국어 (2.6%) - 거의 텍스트 전용
└── 영어 (1.5%) - 중학교 거의 이미지 없음
```

### 4.2 학교급별 특징

| 특징 | 초등학교 | 중학교 |
|------|----------|--------|
| 이미지 비율 | **34.7%** (높음) | 22.6% (낮음) |
| 주된 원인 | 시각적 학습 중시 | 텍스트 기반 심화 |
| 이미지 최다 과목 | 수학 (47.3%) | 과학 (34.4%) |

### 4.3 멀티모달 검색 시사점

1. **수학/과학**: 멀티모달 임베딩 효과 극대화
   - 이미지 정보가 문항 이해에 핵심
   - Image GT 성능 중요

2. **영어/국어**: 텍스트 전용 임베딩으로도 충분
   - 이미지 거의 없음
   - Hybrid GT 성능 중요

3. **초등학교**: 멀티모달 접근 필수
   - 이미지 비율 높음 (34.7%)
   - 시각적 요소가 문항 핵심
