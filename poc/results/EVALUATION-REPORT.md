## 3. 비교 분석

### 3.1 모델별 성능 비교

| 모델 | Image GT Top-5 | Hybrid GT Top-5 | 권장 |
|------|----------------|-----------------|------|
| **2B Multimodal** | **100.0%** | **85.2%** | ✅ **권장** |
| 2B Text-only | 96.3% | 93.4% | - |
| 8B VL (Qwen3VLEmbedder) | 92.6% | 70.5% | ❌ |
| 8B VL (AutoModel) | 71.5% | 65.3% | ❌ |

### 3.2 주요 발견

1. **2B Multimodal이 최고 성능**
   - Image GT에서 100% Top-5 달성
   - 이미지 정보를 효과적으로 활용

2. **8B 모델은 오히려 성능 저하**
   - 보수적 유사도 산출로 인한 구분력 감소
   - 모델 크기와 성능이 비례하지 않음

3. **멀티모달 vs 텍스트 전용**
   - Image GT: 멀티모달 우수
   - Hybrid GT: 텍스트 전용도 경쟁력 있음
